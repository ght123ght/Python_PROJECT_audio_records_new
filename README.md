# Проект на Python: Управление аудиозаписями.

# Описание проекта:
Консольное приложение на языке программирования Python для работы с базой аудиозаписей.
Программа загружает данные из текстового файла и позволяет просматривать и сортировать 
записи по различным критериям с использованием быстрой сортировки (методом Хоара).
Проект выполнен в рамках итоговой индивидуальной работы по дисциплине "Язык программирования Python".

# Входные данные:
Данные хранятся в текстовом файле data.txt.
Одна строка - одна аудиозапись, поля разделены точкой с запятой ";" :
Исполнитель;Трек;Альбом;Год;Длительность;Количество прослушиваний. 

# Функциональные возможности:
Программа предоставляет пользователю циклическое текстовое меню и позволяет:

  1) Просмотреть все аудиозаписи (Сортировка. Исполнитель по возрастанию. Год по убыванию.
     Количество прослушиваний по убыванию).
  2) Просмотреть записи выбранного исполнителя (Сортировка. Альбом по убыванию. Название трека по возрастанию).
  3) Просмотреть записи за заданный период лет (Сортировка. Год по убыванию. Исполнитель по возрастанию).

Предусмотрена обработка ошибок ввода и проверка существования файла.

# Как запустить программу?
  1) Убедитесь, что установлен Python версии 3.1 и выше.
  2) Склонируйте репозиторий или скачайте файлы.
  3) Поместите файл data.txt в ту же папку, что и media_library.py
